As gerações de computadores

(voltar)

Para quem esta começando no mundo da tecnologia, vale a pena iniciar dando aquela voltinha pela historia dos computadores. Fazendo um breve resumo, falaremos das 4 ( quatro) gerações de computadores. A primeira geração de computadores surgiu na década de 1940, os quais funcionavam com válvulas, apresentavam grandes dimensões e os dados eram inseridos através de cartões. Um belo exemplo da época foi o ENIAC. Já os de segunda geração apareceram no final da década de 1950 inicio de 1960. Nesse momento, as máquinas passaram a trabalhar através de transistores e já apresentavam dimensões um pouco menores quando comparados aos da primeira geração. Quando falamos de terceira geração, entramos na era dos circuitos integrados, com aparição dos dispositivos periféricos, esta grande evolução deu-se entre 1965-1970. Finalmente, chegando aos que conhecemos atualmente como PCs, com microprocessadores, memória, dos mais variados tamanhos e também portáteis e com alta capacidade de processamento de dados, estamos nos referindo aos computadores da quarta geração. Esta geração começou em 1970 e vem até os dias atuais.

HTML: como funciona?

(voltar)

Bem, primeiramente vamos esclarecer uma coisinha: html não é uma linguagem de programação.A abreviação HTML refere-se à HyperText Markup Language. Mas e ai? Ainda não entendi, para que serve? O HTML é um arquivo de marcação, o qual possibilita o desenvolvimento de conteúdo para uma pagina web. E através dele, que o seu navegador interpreta o texto, as imagens todo o conteúdo que aparece quando você navega em um website.

Alguns exemplos de Tags

(voltar)

Alguns exemplos de tags Quando criamos uma página html nos utilizamos das tags. Mas o que são tags? Tags são os códigos utilizados para estruturar a página que esta sendo criada. Abaixo segue alguns exemplos de tags:

  • < head > local em que será declarada as informações;
  • < body > tudo que estiver entre o body e o que vai aparecer para o usuário;
  • < i > utilizada para deixar um texto ou palavra em itálico;
  • < strong > para deixar em negrito;
  • < p > para adicionar textos longos, com parágrafos;
  • < h1 > ate < h6 > utilizada para definir títulos e subtítulos;
  • < ul > para gerar uma lista não ordenada;
  • < ol > para gerar uma lista ordenada;
  • < img > utilizada para inserir uma imagem.
